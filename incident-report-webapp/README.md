# Incident Report Log (Web App)

A single-page **front-end** app for logging and filtering operational incidents — built with **vanilla HTML, CSS, and JavaScript** (no framework, no build step). Data persists in the browser via `localStorage`.

## Features
- Add entries with title, category, priority, and notes
- Live **search/filter** and a running stats bar (total + high-priority count)
- Color-coded priority pills; delete entries; sorted newest-first
- Fully responsive, clean UI; **zero dependencies**

## Run it
Open `index.html` in any browser. That's it.

## Files
- `index.html` — structure
- `style.css` — styling (CSS variables, responsive grid/table)
- `app.js` — state, rendering, localStorage persistence, event handling

## Skills demonstrated
DOM manipulation, event handling, client-side state, `localStorage` persistence, responsive CSS, clean separation of structure/style/behavior.
